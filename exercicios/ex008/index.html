Para usar um Marcador

Nesta frase, temos um termo com marcador de texto que destaca uma parte do texto dando uma ênfase descomunal, para uma parte muito importante, usando a tag MARK

Pode-se alterar a cor do marcador usando a tag de css style e utilizando um background-color dentro da tag para algo especifico ou no head para todos os marcadores da página.

Texto grande e pequeno

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Usando a tag DEL

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase que ele foi inserido depois. Usando a tag INS

Texto sobrescrito

Para inserir coisas do tipo x20+3, usando a tag SUP

Texto subescrito

Para inserir coisas do tipo H2O, usando a tag SUB
